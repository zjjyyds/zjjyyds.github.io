报错信息：fatal: unable to access ... Failed to connect to port 443
解决方法：
这是因为网络问题。我使用了手机热点，或者设置了 Git 代理：

问题二：部署后 404

原因：GitHub 仓库设为了 Private（私有）。
解决方法：
进入仓库 Settings -> Danger Zone -> Change visibility，把它改为 Public（公开），然后去 Pages 设置里保存一下就好啦！

4. 常用命令

每次更新博客只需要这三步：

  1. hexo clean (清理缓存)
  2. hexo g (生成网页)
  3. hexo d (部署上传)
